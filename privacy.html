개인정보처리방침

시행일: 2026년 4월 1일

1. 수집하는 정보

개미공탐지수(gongtam.com)는 별도 회원가입 없이 이용 가능하며, 다음 정보만 처리합니다.

  • IP 해시: 채팅 밴 관리 목적으로 IP 주소를 SHA-256 해시로 변환하여 저장합니다. 원본 IP 주소는 서버에 저장되지 않으며, 해시값으로부터 원본 IP를 복원할 수 없습니다.
  • 투표/예측 데이터: 투표 선택과 종가 예측값은 IP 해시와 함께 Firebase에 저장되며, 1일 1회 제한 및 랭킹 표시에 사용됩니다.

2. 로컬 저장소

  • localStorage: 채팅 접힘/펼침 상태, 투표 기록, 종가 예측값 저장
  • sessionStorage: 채팅 닉네임 (탭 종료 시 자동 삭제)

제3자 추적 쿠키는 사용하지 않습니다.

3. 광고

본 서비스는 Google AdSense를 통해 광고를 게재할 수 있습니다. Google은 쿠키를 사용하여 이전 방문 기록을 기반으로 광고를 게재합니다.

4. 데이터 보관 기간

  • 채팅 메시지: 10분 후 자동 만료
  • 투표/예측 데이터: 30일간 보관 후 삭제
  • 밴 기록: 수동 해제 시까지 유지
